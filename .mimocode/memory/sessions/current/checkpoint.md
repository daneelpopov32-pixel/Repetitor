### Discovered

- **DNS propagation on .ru domains**: RIPN nameservers respond but A records may take minutes to hours. Use `nslookup kimstudy.ru 8.8.8.8` to check external propagation.
- **Aeza DNS**: Domain registrar panel allows creating A records. TTL recommended: 3600.
- **Docker daemon may fail to start on fresh Ubuntu 24.04**: Possible causes: conflicting iptables, snap docker, or missing kernel modules. Debug with `journalctl -xeu docker.service`.
- **Heredoc variable expansion pitfall**: Even with `<< 'EOF'` (single-quoted), some shells or copy-paste may lose the quotes. When sending commands to user for manual execution, explicitly warn about `$` variable expansion. Use `\$` escape or write files with `printf`/`tee` instead of heredocs for configs containing shell variables.
