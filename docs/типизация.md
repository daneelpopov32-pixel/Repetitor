Полный набор структурных правил по типам:
Тип 1 — matching. Правый столбец = список годов (4-значные числа). Явного заголовка столбцов в тексте обычно нет.
Тип 2 — sequence (уже реализовано корректно, не трогать).
Тип 3 — matching. В тексте есть буквальный заголовок "ПРОЦЕСС(Ы) (ЯВЛЕНИЕ, СОБЫТИЕ)" и "ФАКТЫ".
Тип 4 — HTML-таблица с пропусками по буквам + отдельный пронумерованный список "пропущенные элементы" ниже таблицы (уже частично реализовано, сохранить).
Тип 5 — matching. Заголовок "ГОСУДАРСТВЕННЫЕ ДЕЯТЕЛИ" или "ДЕЯТЕЛИ", правый столбец — конкретные события/действия, связанные с личностью.
Тип 6 — длинная цитата исторического источника + фраза "выберите ... верные суждения" / "запишите цифры, под которыми они указаны" + пронумерованный список из ~6 утверждений на выбор. НЕ короткий ответ (short_answer) — это отдельный формат "множественный выбор по цитате".
Тип 7 — matching. Заголовок "ПАМЯТНИК(И) КУЛЬТУРЫ", элементы левого столбца — названия в кавычках (произведение/картина) или архитектурные объекты (собор/храм).
Тип 8 — интро "Рассмотрите изображение и выполните задание" + физическое изображение + маркеры ВОВ (1941-1945, соответствующие термины) в тексте вопроса. Уже реализовано верно, сохранить строгую проверку.
Тип 9 — интро "Рассмотрите схему/карту и выполните задание" + физическое изображение карты + вопрос про всё событие/войну целиком ("напишите название войны/события").
Тип 10 — то же интро и изображение карты, но вопрос про конкретный пронумерованный объект на схеме ("город, обозначенный цифрой N").
Тип 11 — изображение карты + цитата текста с пропуском, который нужно заполнить, используя данные карты (комбинация карта+текст).
Тип 12 — изображение карты + список суждений на выбор (структурно как Тип 6, но про карту, не про текстовый источник) — см. исправление типизации выше.
Важно для 9-12: все четыре типа требуют физического изображения — вернуться к решению из Задачи 20 ("карты не требуют изображения для классификации") и отменить его: без картинки контекст этих заданий бессмысленен, значит, наличие images в text_content обязательно для присвоения любой из этих четырёх позиций.
Тип 13 — цитата источника + вопрос про сам источник (атрибуция: название сочинения, факт биографии автора, обстоятельства создания).
Тип 14 — та же структура цитаты, но вопрос про содержание/оценку внутри источника (с чем согласен/не согласен автор, что имел в виду), не про атрибуцию.
Тип 15 — интро "Рассмотрите изображение и выполните задание", одна картинка, вопрос "назовите местность/событие" + обоснование.
Тип 16 — та же база (может быть та же исходная картинка), но дополнительный набор из нескольких пронумерованных изображений-вариантов ("Рис. 1", "Рис. 2"...) для выбора среди них.
Тип 17 — цитаты из воспоминаний (обычно двух фрагментов, А/Б), тематика ВОВ обязательна (жёсткая проверка, как уже реализовано для Типа 8).
Тип 18 — структура ответа строго трёхчастная: "а) причина... б) последствие... в) последствие..." — явный маркер по формату ожидаемого ответа, не по содержанию.
Тип 19 — маркер "раскройте смысл понятия" + "приведите ... факт, конкретизирующий" (определение термина + иллюстрирующий факт).
Тип 20 — маркер "тезис (обобщённое оценочное суждение)" + "два обоснования", каждое из которых должно содержать два факта (сравнение по 2+2 схеме, не путать с Типом 21).
Тип 21 — маркер "приведите аргументы" с явным разделением на именованные категории ответа (например, "Аргумент для России: ___" / "Аргумент для европейской страны: ___") — не просто "аргументируйте", а именно два поименованных аргумента для двух разных объектов сравнения.
